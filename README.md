<h1 align="center">Virtual Museum</h1>
<h4 align="center">PERMM virtual museum on Telegram</h4>

## Идея
Итоговый проект по дисциплине "Основы российской государственности"

Разработан студентами: Поезжаев Леонид, Загидуллина София

Позволяет рассмотреть представленные экспонаты музея "ПЕРММ" на период декабря 2025 года в Telegram.

## Смысл
Сделан для закрытия дисциплины на скорую руку.

Никакой производительности, понятности, или dry кода нет. 

Только множество условных операторов, декораторов и спагетти код
